# UTAU-UST-Alphabet-to-Hangul
로마자로 입력된 UST 가사를 한글로 일괄 변환해주는 플러그인입니다.
이 플러그인은 제미나이 3 Pro를 이용해 작성되었으며, 본인이 일부 수정하고 검수를 마쳤습니다. 

-사용법
1. Release에서 압축파일을 다운받은 뒤 압축해제
2. 폴더를 오픈우타우 설치 경로의 Plugins에 넣는다
3. 오픈우타우 피아노롤 화면 상단의 일괄 수정 -> 레거시 플러그인에서 Romaja to Hangul 을 클릭한다 (보이지 않을 시 새로고침 클릭)

   ---

-옵션 설명

- 단독 자음 앞 노트 병합 (-C + CV)
자음이 어두에 올 시 쓰이는 -C 에일리어스를 뒤 노트에 병합합니다.

- 시작음 (CV) 분리 
"- na" 처럼 적힌 노트를 "- n" "나" 로 변환해줍니다. "- n" "na" 로 적힌 경우는 앞 노트를 보존합니다.

- 모음 연속음(VCV)를 +로 변환
"a a", "eo o", 같은 모음 연속음 노트를 아, 어 로 뒤 모음만 보존해 변환합니다. "e a" 다음 "a a" 가 오는 경우처럼 굳이 한글로 두 번 입력할 필요가 없는 경우 유용합니다.

일부만 변환을 원할 시 변환하고 싶은 노트만 선택 후 플러그인을 실행해주세요.
